A multi-level Pacman, built in **Java (Swing)** using a Maven project structure.
Game Features:
  Ghost:
    1) Smart path selection based on Pacman's position
    2) Different ghost personalities (chase / shy)
    3) Difficulty-based randomness
    4) Respawn delay + spawn grace protection
  
  Power Mode:
    1) Score multiplier for consecutive ghost captures
    2) Power pellets activate frightened state
    3) Visual countdown indicator

  Bonus Life:
    1) Grants extra life
    2) Spawns only in Levels 2 & 3
    3) Limited lifetime

  Difficulty Scaling:
    | Difficulty | Level 1  | Level 2   | Level 3 |
    |------------|----------|-----------|---------|
    | EASY       | 2 ghosts | 3 ghosts | 3 ghosts |
    |------------|----------|-----------|---------|
    | NORMAL     | 3 ghosts | 4 ghosts | 4 ghosts |
    |------------|----------|-----------|---------|
    | HARD       | 4 ghosts | 5 ghosts | 5 ghosts |

  Controls:
    1) Move : Arrow keys
    2) Pause / Resume : P
    3) Menu : Esc
    4) Select difficulty : 1 / 2 / 3 keys
    5) Start / Continue : SPACE
    6) Restart : R
    7) Exit : E
